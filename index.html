Correlations Discovered Between

Poverty and Lack of Healthcare

The data set used for this scatter plot is based on 2014 ACS 1-year estimates:

https://data.census.gov/cedsci/all?q=2014%20ACS&hidePreview=false&tid=ACSDP1Y2014.DP05&y=2014

The current data set incldes data on rates of income, obesity, poverty, etc. by state. This scatter plot was created to show the 2 variables of Healthcare vs. Poverty and how they correlate.

When looking at this scatter plot, click on a circle with the state's abbreviation. A popup will show the correlation between Poverty and the lack of Healthcare. When you move the mouse away the info will disappear.

This project used Javascipt, HTML and CSS. The data was brought in by a CSV file and read by D3
